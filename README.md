# Paranormal Vault 👻

A cinematic, production-ready paranormal investigation platform built with Next.js 14, Supabase, and Framer Motion.

![License](https://img.shields.io/badge/license-MIT-red)
![Next.js](https://img.shields.io/badge/Next.js-14+-black)
![Tailwind](https://img.shields.io/badge/Tailwind-CSS-blue)

## Features
- 🎬 **Cinematic UI:** Ultra-dark theme, atmospheric fog overlays, and smooth parallax effects.
- 📂 **Investigation Archive:** Searchable and filterable database of paranormal cases.
- 🛠️ **Visual CMS:** TipTap-powered rich text editor for building immersive field reports.
- 🔐 **Admin Dashboard:** Command center for content moderation, media management, and analytics.
- 🗺️ **Haunted Explorer:** Dynamic mapping of paranormal hotspots.
- 📱 **Fully Responsive:** Immersive experience across all devices.
- ⚡ **Supabase Backend:** Real-time database, authentication, and secure storage.

## Tech Stack
- **Frontend:** Next.js (App Router), TypeScript, Tailwind CSS, Framer Motion, Shadcn UI.
- **Backend:** Supabase (PostgreSQL, Auth, Storage).
- **Editor:** TipTap Rich Text Engine.
- **Icons:** Lucide React.
